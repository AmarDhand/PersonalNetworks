CODEBOOK FOR PERSONAL NETWORKS



Project contact person: Amar Dhand, adhand@bwh.harvard.edu



Project description: This is R code to process data from REDCap form on personal networks for clinical research. It includes code to create a data table of network metrics and code to create visualizations. This is a supplement of forthcoming paper by Dhand et al. titled ÒA Scalable Online Tool for Quantitative Social Network Assessment Reveals Potentially Modifiable Social Environmental RisksÓ in Nature Communications. 



Instrument: ÒPersonal network survey for clinical researchÓ is a form available in the REDCap Shared Library.

Variable description: Study ID

Variable type: Integer (Unique)

Variable description: Age 

Variable type: Integer
